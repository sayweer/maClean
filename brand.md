# Brand — maClean

_Status: active_

## Direction

maClean uses a warm-monochrome, native macOS utility direction: calm, precise,
and trustworthy. The interface should feel like a careful system tool rather
than an aggressive "cleaner" product.

## Typography

- Use the platform UI font.
- Headings: bold, compact, sentence case.
- Body: regular or medium; never below 12 px.
- Numeric sizes use tabular alignment where the widget supports it.

## Voice

- Turkish, concise, active, and specific.
- Explain safety boundaries before destructive actions.
- Avoid fear-based cleanup language and exaggerated claims.

## Interaction

- Comfortable density by default.
- One primary action per surface.
- No decorative animation; loading motion only when work is actually running.
- Long result sets use a native table and a separate detail panel.
